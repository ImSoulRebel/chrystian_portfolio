      <p>
        <strong>GenUI is an experimental Flutter SDK</strong> that allows AI models

      <p>
        The architecture is elegant and brutal in its simplicity:
      </p>
      <ul>
        <li>
          <strong>Widget Catalog:</strong> you define what visual components the

          <strong>UiAgent:</strong> handles the bidirectional loop. User sends
          message ‚Üí LLM interprets intent ‚Üí generates JSON ‚Üí UI is rendered ‚Üí
          user interacts ‚Üí feedback goes back to LLM.
        </li>
      </ul>
      <p>
        The result: <strong>UI generated in real time, adapted to each user,
        without predefined static screens</strong>.
      </p>
